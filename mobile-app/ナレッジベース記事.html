高圧受電設備の月次点検手順

1. 点検の目的

高圧受電設備の月次点検は、電気事業法に基づく自主保安活動の一環として実施されます。主な目的は以下の通りです。

  • 電気設備の異常を早期に発見し、事故を未然に防止する
  • 設備の劣化状況を把握し、計画的な更新・修繕計画を立案する
  • 法令遵守の証明として記録を残す
  • 設備の安定稼働を確保し、停電リスクを低減する

月次点検は年次点検と比較して簡易的なものですが、重要な異常兆候を発見するための重要な機会です。点検項目を確実に実施し、異常が見つかった場合は速やかに報告・対応することが求められます。

2. 事前準備

点検を効率的かつ安全に実施するために、以下の事前準備を行ってください。

2.1 必要な工具・測定器

  • 絶縁抵抗計（メガー）
  • クランプメーター
  • 検電器
  • 絶縁手袋
  • 絶縁マット
  • ヘルメット
  • 工具セット（ドライバー、レンチ等）
  • デジタルカメラ（スマートフォンでも可）
  • 清掃用具（ブロワー、ウエス等）

2.2 事前確認事項

  • 前回の点検報告書を確認し、指摘事項や経過観察項目を把握する
  • 顧客担当者に点検日時を事前に連絡し、立会いの有無を確認する
  • 停電を伴う点検の場合は、停電範囲と時間を明確にし、顧客の了承を得る
  • 設備の単線結線図を確認し、系統構成を把握する
  • 測定器の校正期限を確認する

測定器の校正期限が切れている場合は、必ず校正済みの測定器を使用してください。校正期限切れの測定器での測定結果は無効となります。

3. 安全対策

高圧電気設備の点検は危険を伴う作業です。以下の安全対策を必ず実施してください。

3.1 基本的な安全対策

  • 必ず2人以上で作業を行う
  • 適切な保護具（絶縁手袋、絶縁マット、ヘルメット）を着用する
  • 作業前に検電器の動作確認を行う
  • 作業範囲に関係者以外が立ち入らないよう区画する
  • 作業前に作業手順と役割分担を確認する

3.2 停電作業時の安全対策

  • 停電操作は必ず手順書に従って実施する
  • 遮断器を開放後、必ず検電を行い、充電されていないことを確認する
  • 検電は複数箇所で実施し、確実に無電圧状態を確認する
  • 接地線を取り付け、作業中の誤通電を防止する
  • 操作禁止札を設置し、誤操作を防止する

重要： 検電を省略したり、一人で高圧電気設備の作業を行ったりすることは絶対に避けてください。感電事故は死亡事故につながる可能性があります。

4. 点検手順

月次点検は以下の手順で実施します。各項目を確実に実施し、異常が見つかった場合は写真撮影と詳細な記録を行ってください。

4.1 外観点検（通電状態）

  • キュービクル周囲の状況確認（物の置き忘れ、水漏れ、小動物の侵入痕等）
  • キュービクル外観の確認（錆、変形、損傷等）
  • 警報装置の動作確認
  • 表示灯の点灯状態確認
  • 異音、異臭、振動の有無確認
  • 温度上昇の有無確認（サーモグラフィーがあれば使用）

キュービクル外観点検の例

4.2 計器類の確認

  • 電圧計の指示値確認（相間電圧、対地電圧）
  • 電流計の指示値確認（各相電流、不平衡率）
  • 電力量計の指示値記録
  • 力率計の指示値確認
  • 各種計器の動作状態確認

4.3 内部点検（必要に応じて停電）

  • 高圧機器の外観確認（変色、損傷、汚損等）
  • 低圧機器の外観確認（変色、損傷、汚損等）
  • 配線の状態確認（変色、損傷、緩み等）
  • 端子部の状態確認（変色、損傷、緩み等）
  • 接地線の状態確認
  • 変圧器の状態確認（油漏れ、異音、温度等）

5. 測定方法

月次点検では、以下の測定を実施します。測定値は前回の値と比較し、大きな変化がないか確認してください。

5.1 負荷電流測定

クランプメーターを使用して、各相の負荷電流を測定します。測定は以下の手順で行います。

  1. クランプメーターの電流レンジを適切に設定する
  2. クランプ部を開き、測定する電線に挟む
  3. 表示値を読み取り、記録する
  4. 各相（R相、S相、T相）で同様に測定する

相間の電流不平衡率が20%を超える場合は、負荷の偏りや相順の誤りが考えられます。詳細な調査が必要です。

5.2 絶縁抵抗測定（停電時）

絶縁抵抗計（メガー）を使用して、高圧回路および低圧回路の絶縁抵抗を測定します。

  • 高圧回路：DC 2500V または 5000V メガーを使用
  • 低圧回路：DC 500V メガーを使用

測定は以下の手順で行います。

  1. 測定対象回路を停電させ、充電されていないことを確認する
  2. 測定対象回路を他の回路から切り離す
  3. メガーの測定レンジを適切に設定する
  4. 測定端子を接続し、測定を開始する
  5. 1分間測定を継続し、指示値を読み取る
  6. 測定値を記録する

絶縁抵抗測定の例

絶縁抵抗値の判定基準：
- 高圧回路：5MΩ以上（基準値）
- 低圧回路：1MΩ以上（基準値）
基準値を下回る場合や、前回測定値から大幅に低下している場合は、詳細な調査が必要です。

6. 報告書作成

点検終了後、以下の内容を含む報告書を作成します。報告書は本アプリの点検報告作成機能を使用して作成してください。

6.1 報告書に含める内容

  • 点検日時、点検者名、立会者名
  • 点検対象設備の基本情報
  • 点検項目と結果（良好/要注意/不良）
  • 測定値（電圧、電流、絶縁抵抗値等）
  • 異常箇所の写真と詳細説明
  • 指摘事項と改善提案
  • 次回点検までの注意事項

6.2 報告書作成のポイント

  • 客観的な事実に基づいて記載する
  • 専門用語を使いすぎず、顧客にも理解しやすい表現を心がける
  • 異常箇所は必ず写真を添付し、何が問題かを明確に説明する
  • 改善提案は優先度を明確にし、概算費用や工期の目安も可能な範囲で記載する
  • 前回からの変化や経年劣化の傾向も記載する

報告書は顧客の設備管理における重要な記録となります。また、法的な証明書としての役割も持ちます。正確かつ丁寧に作成してください。

7. トラブルシューティング

点検中によく遭遇する問題とその対処法を紹介します。

7.1 絶縁抵抗値が低い場合

考えられる原因：

  • 絶縁材の劣化
  • 湿気や結露による影響
  • 汚損や塵埃の蓄積
  • 小動物の侵入による損傷

対処法：

  • 汚損や塵埃がある場合は清掃を行う
  • 湿気が原因の場合は乾燥させる
  • 明らかな損傷がある場合は部品交換を検討
  • 経年劣化の場合は設備更新計画を提案

7.2 異音がする場合

考えられる原因：

  • 機械的な緩み
  • 放電現象
  • 冷却ファンの異常
  • 変圧器のコアの振動

対処法：

  • 緩みがある場合は増し締めを行う
  • 放電音の場合は絶縁不良箇所を特定し対処
  • 冷却ファンの異常は修理または交換
  • 変圧器の異音は専門家による詳細調査を実施

7.3 過熱している場合

考えられる原因：

  • 過負荷状態
  • 接触不良による接触抵抗の増加
  • 冷却システムの不具合
  • 換気不良

対処法：

  • 負荷電流を測定し、定格を超えていないか確認
  • 接触部の増し締めや清掃を実施
  • 冷却システムの点検と修理
  • 換気状態の改善

緊急時の対応： 重大な異常（煙、火花、激しい異音、強い異臭など）を発見した場合は、安全を最優先し、以下の対応を行ってください。
1. 顧客に直ちに報告し、設備の使用停止を依頼
2. 必要に応じて緊急停電措置を実施
3. 上長に報告し、指示を仰ぐ
4. 火災の危険がある場合は消防署に通報
